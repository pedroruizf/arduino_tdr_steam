Librerías. LCD, LM35 y DHT11

Una librería es un conjunto de instrucciones que se han programado para facilitar la programación y control de un sensor o actuador. Vamos a necesitar librerías para elementos como la pantalla LCD, sensor de humedad y temperatura DHT11.

Las librerías que vamos a utilizar las ha recopilado Juanjo López creador de Arduinoblocks en este sitio: http://www.arduinoblocks.com/web/help/libraries. Las descargaremos y descomprimiremos en la carpeta Tu_usuario/Arduino/libraries.
